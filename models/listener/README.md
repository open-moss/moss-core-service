# speaker模型目录

模型可从[此处下载](https://huggingface.co/OpenMOSS/moss-listener-models) 

包含以下文件

- vad.onnx VAD模型
- encoder.onnx 编码器模型
- decoder.onnx 解码器模型
- ctc.onnx CTC模型

vad模型来源：[silero-vad](https://github.com/snakers4/silero-vad)

encoder、decoder、ctc模型来源：[wenet-e2e](https://github.com/wenet-e2e/wenet) - wenetspeech数据集
